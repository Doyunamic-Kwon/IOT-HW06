# 🚗 라즈베리 파이 5 실시간 자동차 감지 시스템 (YOLOv8)

라즈베리 파이 5와 YOLOv8 Nano를 이용해 실시간으로 승용차, 버스, 트럭 등의 차량을 감지하는 프로그램입니다.

## 📌 주요 기능
- 카메라(웹캠)를 통한 실시간 영상 입력 및 차량 인식
- YOLOv8 Nano 기반 승용차, 버스, 트럭 실시간 감지
- 320x240 크기로 빠르게 예측하고, 결과 화면은 640x480으로 2배 키워서 부드럽게 출력

## 📦 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 프로그램 실행
```bash
python main.py
```
